## Dependencies

You will need the following dependencies properly installed on your computer to run the development environment.

* [Node/npm](https://nodejs.org/en/)
* [Axios](https://github.com/axios/axios)
* [MomentJS](https://momentjs.com/)
* [Webpack/Webpack-dev-server](https://webpack.js.org/)

## Installation

* `git clone https://github.com/GSimas/Coronabrasil.git`
* `cd Coronabrasil`
* `Access index.html or run development environment`

## Running / Development

* `yarn dev`
* Visit your app at [http://localhost:8080](http://localhost:8080)
